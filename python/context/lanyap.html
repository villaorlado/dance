Lanyap is a subvariety of the halus character type. It is more vigorous than the luruh character subvariety. In the Yogyakarta tradition, this character is most often called branyak but our primary informant used the name lanyap, which is most commonly associated with the Surakarta tradition. This is not entirely unusual since Sendratari Ramayana dance uses elements from both the Yogyakarta and the Surakarta traditions.
